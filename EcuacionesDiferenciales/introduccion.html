Método de Separación de Variables

El método de separación de variables es una técnica fundamental en el estudio de las ecuaciones diferenciales ordinarias. Este método se utiliza para resolver ecuaciones en las que las variables algebraicamente pueden separarse. Al reorganizar la ecuación diferencial de manera que cada variable y su diferencial correspondiente estén en lados opuestos, se pueden integrar ambos lados de forma independiente. Esto permite obtener una solución general para la ecuación diferencial, facilitando la comprensión y resolución de problemas en diversas áreas de la ciencia y la ingeniería.

Ejercicio 1: Resolver la ecuación diferencial
x ex+y dy dx + y ex+y = 0

1
Primero observamos:

Primero observamos si la ecuación diferencial se puede hacer por el método de variables separables; esto significa que tenga un producto de funciones, que no tenga suma entre las variables dependiente e independiente, y que si tiene sea porque se puede factorizar.

x ex+y dy dx + y ex+y = 0
2
Observamos que hay una suma de variables:

Observamos que hay una suma de variables en el argumento de la función exponencial; sin embargo, esta suma puede separarse aplicando propiedades adecuadas, como se observa a continuación:

x ex ey dy dx = y ex ey dx dy
3
Reescritura:

Reescribiendo la ecuación diferencial vemos que la ecuación se puede separar para que los términos que tienen “y” se agrupen con la diferencial de dy dx y los términos que tienen “x” se agrupen con la diferencial de dx dy , procurando que las funciones y sus diferenciales queden separadas por el símbolo de igualdad

ey / y ey dy dx = ex / x ex dx dy
4
Reducción:

Reducimos la ecuación a su mínima expresión.

∫ dy y = - ∫ dx x
5
Integración:

Integramos del lado de las “y” y del lado de las “x” colocando solo de un lado la constante ya que como la ecuación es de orden uno, solo debemos tener una constante.

lny = lnx + C

Ejercicio 2: Resolver la ecuación diferencial
y2dx - xydy = x2ydy

1
Primero observamos:

Primero observamos si la ecuación diferencial se puede hacer por el método de variables separables esto significa que tenga un producto de funciones, que no tenga suma entre las variables dependiente e independiente, y que si tiene sea porque se puede factorizar.

Comenzamos factorizando el lado izquierdo de la ecuación:

y(ydx - xdy) = x2ydy

Se observa que la variable y al “pasar dividiendo” puede simplificarse.

(ydx - xdy) = x2ydy y

ydx - xdy = x2dy
Identificamos que podemos seguir factorizando términos.
2
A partir de ese paso:

En la ecuación se pueden separar los términos que tienen “y” con la diferencial de “y” y los términos que tienen “x” con la diferencial de “x”.

y dx = x2dy + xdy

ydx=(x2+x)dy
3
Colocamos los términos:

Colocamos los términos que tengan “x” con su diferencial “dx” y colocamos los términos que tengan “y” con su diferencial “dy”.

dx x2 + x = dy y
4
Se integran ambos lados:

Se integra de ambos lados colocando solo de un lado la constante ya que como la ecuación es de orden uno, solo debemos tener una constante.

∫ dx x2 + x = ∫ dy y

La integral del lado izquierdo se puede reescribir de la siguiente manera y la del lado derecho es una integral directa que resolvemos.

∫ dx x2( 1 x + 1) = ln(y)+C

Para resolver la integral con respecto a ”x” se usa cambio de variable.

u = 1 x + 1,   du = - 1 x2 dx
∴   - ∫ 1 u   du = ln(y) + C
- ln(u) = ln(y) + C
- ln ( 1 x + 1 ) + C = ln(y) + C

Se puede notar que aparece una constante para cada integral indefinida, sin embargo al sumarlas se puede reducir a una sola.

-ln ( 1 x + 1 ) = ln(y) + C

Además, a la constante se puede reescribir como un logaritmo de la constante y seguir siendo constante, con el objetivo de utilizar propiedades de los logaritmos y simplificar la solución.

ln ( 1 x + 1 )-1 = ln(y) + lnC
ln ( 1 1 x + 1 ) = ln(y) + lnC
ln ( 1 1 x + x x ) = ln(y) + lnC
ln ( 1 1 + x x ) = ln(y) + lnC
ln ( x x + 1 ) = ln(y) + lnC

Se le puede aplicar exponencial a toda la ecuación.

eln( x x + 1 ) = eln(y) + lnC

Finalmente, se obtiene la solución general a la ecuación diferencial.

x x + 1 = y + C
